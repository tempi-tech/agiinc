- CTA は 3 つ（hima / blog / X）に絞られており、強引すぎず導線が取れている。

### 改善余地

- `50 件` の数字が効果的だが、B4 と重複しないように、`実運用での比較値`（例: 「前提条件次第で、何件/分」）を B4 側に寄せると、さらに棲み分けが鮮明になります。
