# デモ GIF シーンスクリプト

> Creative Director レア・デュボワ | 2026-02-14 | ステータス: ドラフト

---

## 概要

本書は `demo-assets-spec.md` セクション 1 に定義された 1-A（LP 用）と 1-B（X 投稿用）のデモ GIF 制作用シーンスクリプトである。各シーンのマウス操作・クリック位置・入力テキスト・待機時間を詳細に記述する。

### 前提

- ブラウザ: Chrome、ウィンドウ幅 1440px（キャプチャ後にリサイズ）
- 画面: Hima ワークスペース（`hima.agiinc.io`）
- ダミーデータ: `docs/hima/demo-data.csv`（商品説明文生成ユースケース、10 件）
- マウスカーソルは常に滑らかに移動（急な移動を避ける）
- クリック時: カーソル周囲に薄い円形ハイライト（`rgba(45,58,140,0.15)`, 半径 20px, 0.3s フェード）
- テキスト入力: タイプライター風（1 文字ずつ、50ms/文字）
- シーン間遷移: 0.3s クロスフェード

## 付録: 演出チェックリスト

撮影・編集時に以下を確認すること:

- [ ] マウスカーソルの動きが滑らかか（急な加速・停止がないか）
- [ ] クリック時のハイライトエフェクトが全箇所で発動しているか
- [ ] テキスト入力がタイプライター風に見えるか
- [ ] 変数検出のリアルタイム更新が視覚的に明確か
- [ ] 進捗バーのアニメーションが滑らかか
- [ ] シーン間のクロスフェードが 0.3s で統一されているか
- [ ] エンドフレームのフォント・サイズがスペック通りか
- [ ] 全体の尺が仕様の範囲内か（1-A: 15-20s / 1-B: 25-30s）
- [ ] ダミーデータがリアルに見えるか（モックデータとわからないレベル）
