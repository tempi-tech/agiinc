### 6-2. 投稿方法: X intent URL によるスレッド投稿

現在の運用方法（`x-operations.md`: intent URL 経由）に準拠し、以下の手順で投稿する。

**手順:**

1. **T1 投稿**: intent URL でツイート 1 を投稿
2. **T1 の tweet_id を取得**: 投稿完了後、URL から tweet_id を取得
3. **T2 投稿**: T1 への返信として intent URL で投稿（`in_reply_to` パラメータ使用）+ A4 画像を手動添付
4. **T3-T5**: 同様に直前のツイートへの返信として順次投稿。各ツイートに対応アセットを添付

**投稿予約の代替案（推奨）:**

X Premium+ の予約投稿機能を使用する場合:
1. **3/24 13:00**: TweetDeck or X 公式予約機能でスレッド全 5 ツイートを入稿
2. 各ツイートにアセットを添付（T2: A4, T3: A5, T5: A1）
3. 投稿予約日時を **3/25 12:00 JST** に設定
4. **3/24 14:00**: プレビュー確認（テキスト・画像・GIF・リンクの表示チェック）
5. **3/25 11:45**: 予約状態の最終確認

**障害時のフォールバック:**
- 予約投稿が失敗した場合、手動投稿に切り替え（下書きテキスト + アセットをローカル保存しておく）
- 手動投稿の所要時間: 5 分以内
