# デモ GIF 1-B（X投稿用）制作レポート

> Creative Director レア・デュボワ | 2026-02-15 | ステータス: v1 制作完了

---

## 1. 実施概要

デモ GIF 1-B（X 投稿用、〜28 秒）を Remotion パイプラインで制作した。1-A v2 で確立したクロスフェード・カスタムフォント・CSV ドラッグ演出のコードベースを拡張し、5 シーン + エンドフレーム構成の X 向けデモ動画を生成。ffmpeg パイプラインによる GIF/WebM 変換まで完了した。

### 制作環境

| 項目 | 値 |
|---|---|
| Remotion | 4.0.259 |
| Node.js | v25.5.0 |
| ffmpeg | 7.1.1 |
| 解像度 | 1200 × 675px (16:9) |
| FPS | 12 |
| フレーム数 | 336 |
| 尺 | 28.0s |

## 5. ビジュアル品質評価

### 5-1. 1-A v2 から継承した品質

- クロスフェード遷移（`@remotion/transitions` + `fade()`）
- カスタムフォント（Inter / Noto Sans JP / JetBrains Mono）
- CSV ドラッグアニメーション（浮遊アイコン + ドロップゾーンハイライト）
- クリックハイライト（不透明度 0.22）
- バッチ実行ボタン表示
- ブランドカラー一貫性

### 5-2. 1-B 固有の品質

- **S0 全景ショット**: 3 パネル構成が一目で把握できる静止ショット
- **レシピ名入力**: タイプライター風で「商品説明文ジェネレータ」が自然に入力される
- **システムプロンプト接頭辞**: 「あなたはプロのコピーライターです。」が入力され、変数含みプロンプトへ続く
- **CSV データプレビュー**: ヘッダー + 3 行でデータのリアリティを補強
- **バッチ 0.5s リズム**: 1-A (0.33s) より間を取り、X タイムラインでのスクロール閲覧に適した速度感
- **2 行展開**: 行 1 と行 3 を順次展開し、結果の多様性を見せる
- **エンドフレーム**: チャコール背景 + 白文字で URL と CTA が明確

## 10. まとめ

- **制作方式**: Remotion モックアップ（1-A v2 と同一パイプライン）
- **ファイルサイズ**: GIF 1.6 MB / WebM 979 KB。仕様上限の 32% / 49% で品質向上余地あり
- **尺**: 28.0s（仕様範囲 25-30s 内）
- **シーン構成**: S0 全景 → S1 レシピ → S2 CSV → S3 バッチ → S4 エクスポート → End CTA
- **ブロッカー**: なし
