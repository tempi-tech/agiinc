# Hima デモアセット仕様書

> Creative Director レア・デュボワ | 2026-02-14 | ステータス: CMO レビュー反映済み（v2）

---

## 概要

Hima ローンチ（2026-03-25）に必要なビジュアルアセットを一覧化し、仕様・制作順・納品日を定義する。全アセットは LP デザイン方針書（`docs/hima/lp-design.md`）のブランドアイデンティティに準拠する。

### ブランド準拠ルール（全アセット共通）

- カラー: オフホワイト `#FAFAF8` / チャコール `#1A1A1A` / ディープインディゴ `#2D3A8C` 基調
- フォント: 日本語 Noto Sans JP / 英語 Inter / コード JetBrains Mono
- トーン: クリーン、余白が多い、AI 装飾なし（パーティクル・ロボット・ニューラルネット図禁止）
- 画像形式: WebP（静止画）/ GIF or WebM（動画）

## 1. デモ GIF 仕様

LP Section 4、X 投稿（03/19）、YouTube V2 の素材を兼ねるメインアセット。Hima の実画面を操作キャプチャして制作する。

### 1-A. LP 用デモ GIF（メイン）

| 項目 | 仕様 |
|---|---|
| 用途 | LP Section 4「デモ GIF / 動画」 |
| サイズ | 1200 × 750px（16:10）|
| ファイル形式 | GIF（フォールバック）+ WebM（メイン表示） |
| ファイルサイズ上限 | GIF: 5MB / WebM: 2MB |
| 長さ | 15〜20 秒（ループ再生） |
| フレームレート | 15fps |
| 表示方式 | 自動再生 + ループ、操作バー非表示 |
| 枠の演出 | `box-shadow: 0 8px 32px rgba(0,0,0,0.12)` で浮遊感 |
| 背景 | ウォームグレー `#F0EEEB` の上に配置 |

**シーン構成（4 シーン・各 4-5 秒）:**

| # | 秒数 | 内容 | キャプチャ対象画面 | 対応ショット |
|---|---|---|---|---|
| S1 | 0-4s | レシピ作成 | プロンプト欄に `{{商品名}}` `{{カテゴリ}}` を入力。変数リスト自動検出 | YouTube S-02 |
| S2 | 4-9s | CSV 読み込み | CSV ドラッグ＆ドロップ → カラムマッピング完了 | YouTube S-04, S-05 |
| S3 | 9-15s | バッチ実行 | 「バッチ実行」押下 → 進捗バー更新 → 完了 | YouTube S-06 |
| S4 | 15-20s | 結果エクスポート | 結果テーブル確認 → 「CSV エクスポート」押下 | YouTube S-07, S-08 |

**演出ルール:**
- マウスカーソルの動きは滑らかに、急な移動を避ける
- クリック時にカーソル周囲に薄い円形ハイライト（`rgba(45,58,140,0.15)`, 半径 20px, 0.3s フェード）
- シーン間は 0.3s のクロスフェードで切り替え
- テキスト入力はタイプライター風（1 文字ずつ表示）

### 1-B. X 投稿用デモ GIF

| 項目 | 仕様 |
|---|---|
| 用途 | X 投稿 `2026-03-19-lp-release-demo-gif.md` |
| サイズ | 1200 × 675px（16:9、X 推奨比率） |
| ファイル形式 | GIF |
| ファイルサイズ上限 | 15MB（X 上限） |
| 長さ | 25〜30 秒 |
| フレームレート | 12fps（ファイルサイズ制約） |

**シーン構成（5 シーン）:**

LP 用の 4 シーンに加え、冒頭にワークスペース全景（2 秒）を追加。

| # | 秒数 | 内容 |
|---|---|---|
| S0 | 0-2s | ワークスペース全景。左レシピエディタ + 右入力エリア + 下結果テーブル |
| S1 | 2-8s | レシピ作成（変数入力 + 自動検出） |
| S2 | 8-14s | CSV ドラッグ＆ドロップ + マッピング |
| S3 | 14-22s | バッチ実行（進捗バー・完了件数・行ステータス更新） |
| S4 | 22-28s | 結果テーブル確認 + CSV エクスポート |
| End | 28-30s | エンドフレーム: `hima.agiinc.io` + 「無料・登録不要」 |

**エンドフレーム仕様:**
- 背景: チャコール `#1A1A1A`
- テキスト: 白 `#FFFFFF`
- URL `hima.agiinc.io` を Inter Bold 24px で中央配置
- サブテキスト「無料・登録不要・BYOK」を Inter Regular 14px で URL 下 8px に配置

### 1-C. YouTube V2 用画面キャプチャ素材

YouTube V2 台本（`docs/hima/youtube-v2-script.md`）のショット S-01〜S-09 に対応する画面キャプチャ。デモ GIF の素撮りから抽出し、Remotion で動画編集時に使用する。

| ショット | 内容 | 形式 | 解像度 |
|---|---|---|---|
| S-01 | ワークスペース全景 | WebM（5s） | 1920 × 1080 |
| S-02 | プロンプト入力 + 変数自動検出 | WebM（10s） | 1920 × 1080 |
| S-03 | モデル選択 + レシピ保存 | WebM（5s） | 1920 × 1080 |
| S-04 | CSV ドラッグ＆ドロップ | WebM（5s） | 1920 × 1080 |
| S-05 | カラムマッピング + 件数確認 | WebM（5s） | 1920 × 1080 |
| S-06 | バッチ実行中（進捗バー + ステータス更新） | WebM（15s） | 1920 × 1080 |
| S-07 | 結果テーブル + 行展開 | WebM（8s） | 1920 × 1080 |
| S-08 | CSV エクスポート | WebM（5s） | 1920 × 1080 |
| S-09 | エンドカード（別途 Remotion で生成） | — | — |

**キャプチャ環境:**
- ブラウザ: Chrome、ウィンドウ幅 1440px
- ブラウザ UI（タブバー・アドレスバー）は非表示にキャプチャ
- ダミーデータ: 「商品説明文生成」のユースケース。商品名 10 件程度の CSV を使用
- 画面には実データが表示されること。モックアップ不可

## 6. 制作ツール・パイプライン

| 工程 | ツール | 備考 |
|---|---|---|
| 画面キャプチャ | Chrome DevTools + OBS or Kap | ブラウザ UI 非表示でキャプチャ |
| GIF 編集 | ffmpeg（WebM → GIF 変換 + リサイズ + フレームレート調整） | `ffmpeg -i input.webm -vf "fps=15,scale=1200:-1" -loop 0 output.gif` |
| 静止画制作 | HTML/CSS → スクリーンショット or Nano Banana（画像生成スキル） | ブランド準拠を保つため HTML/CSS ベースを推奨 |
| YouTube 動画 | Remotion（React ベース動画生成） | スキル `remotion` を使用 |
| ナレーション | ElevenLabs TTS | スキル `elevenlabs-tts` を使用 |
| 表示確認 | Chrome Browser スキル | 公開後の見た目チェック |
