## A2: カウントダウン画像

### 使用先

| 項目 | 内容 |
|---|---|
| 投稿 | D3 `2026-03-21-launch-minus-4-countdown.md` |
| 投稿日 | 2026-03-21（土）12:00 JST |
| カテゴリ | ビルドインパブリック |
| 投稿文脈 | ローンチ4日前。「あと4日」の緊張感を視覚的に補強する |

### デザイン仕様

| 項目 | 仕様 |
|---|---|
| サイズ | 1200 x 675px |
| 形式 | PNG |
| 背景 | チャコール `#1A1A1A` |
| メイン要素 | 数字「4」— Inter Bold 120px, 白 `#FFFFFF`, 画面中央 |
| サブテキスト1 | 「days to Hima」— Inter Regular 20px, `#6B6B6B`, 数字下 16px |
| サブテキスト2 | `hima.agiinc.io` — Inter Regular 14px, `#6B6B6B`, サブテキスト1 下 8px |

### デザイン意図

- ダークな背景に白い大きな数字で「カウントダウン」の視覚的インパクトを出す
- 余白を活かしたミニマルなレイアウト。情報は3要素のみ
- 投稿テキスト「ローンチまで、あと4日。」と視覚的に連動する
- X のタイムラインで目を引くコントラスト（暗い背景は TL で目立つ）

### 制作方法

HTML/CSS → スクリーンショットを推奨（ブランド準拠を担保するため）

## A3: ティーザー画像

### 使用先

| 項目 | 内容 |
|---|---|
| 投稿 | D4 `2026-03-24-launch-eve-teaser.md` |
| 投稿日 | 2026-03-24（火）12:00 JST |
| カテゴリ | プロダクト価値訴求 |
| 投稿文脈 | ローンチ前夜。「明日、出します。」の一言で翌日 3/25 への期待感を最大化する |

### デザイン仕様

| 項目 | 仕様 |
|---|---|
| サイズ | 1200 x 675px |
| 形式 | PNG |
| 背景 | オフホワイト `#FAFAF8` |
| メイン要素 | テキストロゴ「Hima」— Inter Bold 48px, チャコール `#1A1A1A`, 画面中央やや上 |
| コピー | 「明日、出します。」— Noto Sans JP Bold 24px, チャコール `#1A1A1A`, ロゴ下 24px |
| 日付 | 「2026.03.25」— Inter Regular 16px, `#6B6B6B`, コピー下 12px |

### デザイン意図

- A2 のダーク背景から一転、ライトな背景で「新しい朝」の印象を出す
- 投稿テキスト冒頭「明日、出します。」と画像のコピーが完全に呼応する
- 日付を入れることで「いつ」が一目で伝わる。フォロワーのカレンダーに残る
- ロゴが正式に決まっていない場合はテキストロゴ「Hima」で代用。Inter Bold のシンプルな表記がブランドトーンに合う

### 制作方法

HTML/CSS → スクリーンショットを推奨
