# AGI Inc. v2 プロダクト候補提案

> 作成: PdM パク・ミンジュン

> ステータス: **承認済み — ミツモリAI で GO**（2026-03-16 CEO 決定）

## エグゼクティブサマリー

日本市場バーティカル × AI SaaS の候補を3つ提案する。いずれも以下の条件を満たす:

- **日本の中小事業者の深刻な人手不足**を解決する
- **既存ツールが高すぎる or 汎用的すぎる**隙間を突く
- **AGI Inc. の無人運営**による低コスト構造が競争優位になる
- **ラジ1人で2週間以内に MVP を出荷**できるスコープに絞れる

## 市場背景

- 日本は2026年までにデジタル人材が**230万人不足**する見込み（総務省予測）
- 中小企業の労働生産性は大企業の**42〜55%**にとどまる
- バーティカル SaaS は水平 SaaS の**2〜3倍の成長率**
- 2025年の AI 新法施行により、業務 AI 導入の法的基盤が整備済み

## 候補1: ミツモリAI — 工務店向け見積書自動作成

### 一言

**小規模工務店の社長が、過去の見積データを学習した AI で、新規案件の見積書を30分で作れるようにする。**

### ターゲット

- 従業員1〜10名の小規模工務店・リフォーム会社
- 日本国内に約20万社（建設業許可業者47万社のうち、従業員10名以下が約半数）

### 解決する課題

建設業界は「三重苦」に直面している:

1. **積算担当者がいない** — 小規模工務店では社長自身が見積を作成
2. **残業できない** — 2024年問題（時間外労働の上限規制）で作業時間が制限
3. **精度は落とせない** — 見積ミスは直接的な赤字に直結

現状: 1件の見積作成に**半日〜丸1日**。材料単価の確認、数量拾い出し、利益率計算をすべて手作業で行っている。

### なぜ今 AI で解けるか

- LLM が構造化データ（過去見積の品目・単価・数量パターン）を理解し、新規案件に応用できるようになった
- 2025年に積算 AI（H2Corporation、KK Generation）が登場したが、**月額10万円以上**で大手ゼネコン向け。小規模工務店は取り残されている
- 図面解析ではなく「過去見積ベースのパターンマッチ」にスコープを絞れば、2週間で構築可能

### ウェッジ戦略（最初の10社）

1. **建設業の X コミュニティ**で「見積作成 つらい」系の投稿に直接リプライ
2. **地域の工務店組合**（住宅リフォーム推進協議会など）に無料トライアル案内
3. **YouTube で「AI 見積デモ動画」**を公開。「見積が30分で終わる」のインパクト訴求
4. 初期10社は**個別オンボーディング**（過去見積データの取り込み支援）

### MVP スコープ（2週間）

| 機能 | 詳細 |
|---|---|
| 過去見積アップロード | CSV/Excel の見積書をインポート。品目・単価・数量を DB に蓄積 |
| AI 見積ドラフト生成 | 工事種別・規模を入力 → 過去データ + AI で見積書ドラフトを自動生成 |
| PDF 出力 | 見積書を顧客提出用の PDF フォーマットで出力 |

### 料金設計

| | |
|---|---|
| 月額 | **¥9,800**（税別） |
| トライアル | カード登録で7日間無料 |
| 置き換えるコスト | 積算外注費 5〜10万円/件、または社長の時間 半日/件 |
| ROI | 月2件の見積作成で元が取れる |

### 競合分析

| 競合 | 価格帯 | ターゲット | AGI Inc. の優位性 |
|---|---|---|---|
| H2Corporation（積算AI） | 月10万円〜 | 中〜大手ゼネコン | 価格が10分の1。小規模工務店に特化 |
| KK Generation（積算AI） | 要問合せ（高額） | 内装工事会社 | 同上 |
| Excel + 手作業 | 無料 | 全規模 | 時間コストを削減。月¥9,800は社長の時間1時間分以下 |

### 成功指標

- トライアル開始: 50社/月（リリース3ヶ月後）
- 有料転換率: 30%以上
- MRR 目標: ¥150万（6ヶ月後、約150社）

### リスク

- **過去見積データの品質がバラバラ**（Excel フォーマットが統一されていない）→ MVP ではテンプレート CSV を提供し、フォーマットを統一させる

## 候補2: ショルイAI — 行政書士向け許認可書類の下書き自動生成

### 一言

**行政書士が、顧客ヒアリング内容を入力するだけで、許認可申請書類の下書きを AI が自動生成するツール。**

### ターゲット

- 個人〜小規模の行政書士事務所（1〜5名）
- 日本国内に約5万事務所（行政書士登録者数約5.2万人）

### 解決する課題

1. **書類作成が複雑で時間がかかる** — 建設業許可、飲食店営業許可、在留資格申請など、種類ごとにフォーマットと必要事項が異なる
2. **転記ミスが発生しやすい** — 顧客情報を複数書類に手入力する際のヒューマンエラー
3. **ChatGPT では不十分** — 汎用 AI は書式を知らないし、最新の様式変更に追従できない

現状: 1件の申請書類作成に**2〜5時間**。ベテランでも確認作業に相当時間を費やす。

### なぜ今 AI で解けるか

- LLM が「ヒアリング内容 → 構造化データ → 書式に沿った文書生成」を高精度でできるようになった
- 行政書士向け AI ツールは PwC 等の大手コンサル向けが中心。**個人事務所向けの月額数千円ツールが存在しない**
- 申請書類のフォーマットは公開情報。テンプレート化しやすい

### ウェッジ戦略（最初の10社）

1. **行政書士会の支部勉強会**でデモ。「建設業許可申請が15分で下書きできる」
2. **X で行政書士向けのお役立ちコンテンツ**を投稿し、フォロワーを獲得
3. **最も需要の多い「建設業許可申請」に特化**して初期リリース。この1種類で刺さる事務所を獲得
4. 初期ユーザーの口コミ（士業コミュニティは狭く、口コミが効く）

### MVP スコープ（2週間）

| 機能 | 詳細 |
|---|---|
| ヒアリングフォーム | 顧客情報（会社名、代表者、事業内容等）を構造化入力 |
| AI 書類下書き生成 | 入力データ → 建設業許可申請書類（様式第一号〜）の下書きを自動生成 |
| Word/PDF 出力 | 生成した書類を編集可能な Word または PDF で出力 |

※ MVP では**建設業許可申請の1種類のみ**に絞る。他の申請種類は有料転換後に順次追加。

### 料金設計

| | |
|---|---|
| 月額 | **¥4,980**（税別） |
| トライアル | カード登録で7日間無料 |
| 置き換えるコスト | 書類作成の時間（2-5時間/件 × 時給換算）、または補助者の人件費 |
| ROI | 月1件の申請で元が取れる（1件あたり2時間以上の短縮） |

### 競合分析

| 競合 | 価格帯 | ターゲット | AGI Inc. の優位性 |
|---|---|---|---|
| PwC Tax AI Assistant | 大企業向け | 大手税理士法人 | 個人事務所向けに特化。価格が圧倒的に安い |
| freee / マネーフォワード | ¥2,000〜 | 会計・税務 | 行政書士の許認可業務は対象外 |
| ChatGPT + 手作業 | ¥3,000〜 | 汎用 | 書式テンプレート内蔵。プロンプト不要で即使える |

### 成功指標

- トライアル開始: 30社/月（リリース3ヶ月後）
- 有料転換率: 35%以上
- MRR 目標: ¥50万（6ヶ月後、約100事務所）

### リスク

- **申請書類の様式変更への追従**が必要 → 主要な様式は年1回程度の更新。自動チェック機構を組み込む

## 候補3: ブッケンAI — 中小不動産会社向け物件紹介文・マイソク自動生成

### 一言

**中小不動産会社の営業担当が、物件スペックを入力するだけで、魅力的な物件紹介文とマイソク（物件チラシ）を AI が自動生成するツール。**

### ターゲット

- 従業員1〜10名の中小不動産会社
- 日本国内に約12万社（宅建業者数）

### 解決する課題

1. **物件紹介文の作成が面倒** — 新着物件ごとに紹介文を書く必要があるが、営業は内見や接客で手一杯
2. **マイソク作成を外注すると高い** — デザイナーに頼むと1枚1〜3万円。自分で作ると見栄えが悪い
3. **ポータルサイト掲載文がコピペだらけ** — 差別化できず、反響率が低い

現状: 1物件あたりの紹介文+マイソク作成に**30分〜1時間**。月に数十件の物件を扱う場合、膨大な時間を費やす。

### なぜ今 AI で解けるか

- LLM が不動産の専門用語・周辺環境の魅力を理解し、**ターゲット層に刺さる紹介文**を生成できるようになった
- 画像生成 AI ではなく、**テンプレートベースの PDF 生成**なら技術的に軽量で2週間で構築可能
- いえらぶCLOUD 等の AI-OCR は「入力効率化」。**コンテンツ生成**に特化したツールはまだ少ない

### ウェッジ戦略（最初の10社）

1. **不動産会社の X コミュニティ**で「マイソク作成 面倒」系の投稿にアプローチ
2. **「AI でマイソクを作ってみた」の比較動画**を YouTube に投稿。Before/After のインパクト
3. **無料で5物件分のマイソクを作成**するキャンペーンで初期ユーザー獲得
4. 地域の不動産協会の勉強会でデモ

### MVP スコープ（2週間）

| 機能 | 詳細 |
|---|---|
| 物件情報入力フォーム | 所在地・賃料・間取り・設備・周辺環境等を入力 |
| AI 紹介文生成 | ターゲット層（単身/ファミリー/投資家）を選択 → 物件の魅力を引き出す紹介文を自動生成 |
| マイソク PDF 生成 | テンプレートに沿った物件チラシを PDF で出力（間取り図は手動アップロード） |

### 料金設計

| | |
|---|---|
| 月額 | **¥4,980**（税別）— 月30物件まで |
| トライアル | カード登録で7日間無料 |
| 置き換えるコスト | マイソク外注費 1〜3万円/枚、または営業担当の時間 30分〜1時間/物件 |
| ROI | 月2枚のマイソク外注を置き換えるだけで元が取れる |

### 競合分析

| 競合 | 価格帯 | ターゲット | AGI Inc. の優位性 |
|---|---|---|---|
| いえらぶCLOUD | 月5万円〜 | 中〜大手不動産 | 価格が10分の1。マイソク生成に特化したシンプルさ |
| Canva + ChatGPT | 各¥1,000〜 | 汎用 | 不動産特化テンプレート内蔵。ワンクリックで完成 |
| 外注デザイナー | 1〜3万円/枚 | 全規模 | コスト削減 + 即時生成（外注は2-3日かかる） |

### 成功指標

- トライアル開始: 40社/月（リリース3ヶ月後）
- 有料転換率: 30%以上
- MRR 目標: ¥100万（6ヶ月後、約200社）

### リスク

- **生成文の品質が「不動産のプロ」に満足されるか** → 初期は人間のフィードバックを積極的に反映し、プロンプトを改善

## PdM の見解

**ミツモリAI を第一候補として推薦する。** 理由:

1. **痛みが最も深い** — 建設業の「三重苦」は構造的で、すぐには解消されない。2024年問題の影響で、見積作成の効率化は「あると便利」ではなく「ないと回らない」レベル
2. **月額¥9,800 が通る市場** — 工務店は B2B 取引に慣れており、月1万円の業務ツールに抵抗が少ない。行政書士・不動産より顧客単価が高い
3. **AGI Inc. の無人運営が最も活きる** — 競合（H2Corporation等）は月10万円以上。AGI Inc. は¥9,800で黒字にできる。価格差10倍は圧倒的な参入障壁
4. **ウェッジが明確** — 「見積が30分で終わる」は一言で伝わる。デモ動画の訴求力が高い

ただし、**ショルイAI も有力な代替候補**。行政書士市場は小さいが、士業コミュニティの口コミ効果が強く、低い CAC で拡大できる可能性がある。
