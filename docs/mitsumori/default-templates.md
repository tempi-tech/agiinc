# ミツモリAI — デフォルト見積テンプレート

> 過去見積データがないユーザーへのフォールバック用。AI 見積生成時に、該当カテゴリの過去データが 0 件の場合、このテンプレートを参考データとして LLM に渡す。

## データ仕様

各テンプレートの項目は `estimate_items` テーブルのカラムに準拠:

| フィールド | 対応カラム | 説明 |
|---|---|---|
| 品目名 | name | TEXT |
| 仕様 | specification | TEXT |
| 数量 | quantity | REAL |
| 単位 | unit | TEXT |
| 単価（円） | unit_price | INTEGER |
| 金額（円） | amount | INTEGER |
| 表示順 | item_order | INTEGER（1始まり） |

※ 単価・金額はすべて税抜。概算であり、AI が工事条件に応じて調整する前提。

## 実装メモ

### テンプレートの利用タイミング

```
AI 見積生成リクエスト
  ↓
historical_estimates から同カテゴリの過去データを検索
  ↓
  0 件 → このテンプレートを参考データとして LLM に渡す
  1 件以上 → 過去データを優先して渡す（テンプレートは使わない）
```

### category マッピング

| テンプレート名 | category 値 | 備考 |
|---|---|---|
| 外壁塗装 | exterior | |
| キッチンリフォーム | interior | 水回り要素あるが spec では interior に分類 |
| 浴室リフォーム | plumbing | |
| 屋根修理 | exterior | |
| 内装工事 | interior | |

### 単価の根拠

- 2025〜2026年の住宅リフォーム市場の一般的な施工単価を参考
- 地域差（都市部 vs 地方）は考慮していない。AI が工事条件に応じて調整する前提
- 資材高騰等の影響は定期的に見直す必要あり
