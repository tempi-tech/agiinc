# ミツモリAI プライバシーポリシー

最終更新日: 2026年3月19日

AGI Inc.（以下「当社」）は、「ミツモリAI」（以下「本サービス」）における個人情報の取り扱いについて、以下の通りプライバシーポリシーを定めます。

---

## 1. 収集する情報

当社は、本サービスの提供にあたり、以下の情報を収集します。

### 1.1 ユーザーが提供する情報

| 情報の種類 | 収集するタイミング | 具体例 |
|---|---|---|
| アカウント情報 | 利用登録時 | メールアドレス、パスワード（ハッシュ化して保存）、氏名 |
| 会社情報 | 利用登録時・設定変更時 | 会社名、郵便番号、住所、電話番号、適格請求書発行事業者番号 |
| 見積データ | サービス利用時 | CSV でアップロードされた過去の見積データ（案件名、工事種別、品目、単価、金額等） |
| 見積入力情報 | サービス利用時 | AI 見積生成時に入力された工事条件（案件名、顧客名、工事概要、面積、予算等） |

### 1.2 自動的に収集される情報

| 情報の種類 | 説明 |
|---|---|
| アクセスログ | IP アドレス、アクセス日時、ブラウザの種類、参照元 URL |
| Cookie 情報 | セッション管理および認証状態の維持に使用（詳細は第5条参照） |
| 決済関連情報 | Stripe を通じて処理されるため、クレジットカード番号を当社が直接保存することはありません |

## 2. 情報の利用目的

収集した情報は、以下の目的でのみ利用します。

1. **本サービスの提供・運営**: アカウント管理、認証、見積生成、PDF 出力
2. **AI 見積生成の精度向上**: ユーザーがアップロードした過去見積データを、当該ユーザーの見積生成にのみ利用（他のユーザーのデータと混合しません）
3. **課金処理**: 月額利用料金の請求、トライアル管理
4. **サービスの改善**: 利用状況の分析、機能改善の検討
5. **お知らせ・サポート**: サービスに関する重要な通知、お問い合わせへの対応

## 3. 第三者への情報提供

当社は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

### 3.1 業務委託先

| 委託先 | 提供する情報 | 目的 |
|---|---|---|
| **Cloudflare, Inc.** | アクセスログ、コンテンツデータ | 本サービスのホスティング、CDN、データベース（D1）、ファイルストレージ（R2） |
| **Stripe, Inc.** | メールアドレス、決済情報 | 月額利用料金の決済処理、サブスクリプション管理 |
| **Anthropic, PBC** | 見積データ（匿名化処理済み） | AI による見積ドラフト生成のための API リクエスト |

### 3.2 法令に基づく場合

法令に基づく開示要請を受けた場合、裁判所の命令がある場合、またはユーザーもしくは第三者の生命・身体・財産の保護のために必要な場合は、個人情報を開示することがあります。

## 4. データの保存・管理

### 4.1 保存場所

ユーザーデータは、Cloudflare のインフラストラクチャ上に保存されます。

### 4.2 保存期間

| データの種類 | 保存期間 |
|---|---|
| アカウント情報 | アカウント存続中および解約後30日間 |
| 見積データ（過去見積・AI 生成見積） | アカウント存続中および解約後30日間 |
| アクセスログ | 最大90日間 |
| 決済履歴 | 法令に定める保存義務期間（最大7年間） |

### 4.3 データの削除

- ユーザーは、本サービスの設定画面からアップロードした見積データを随時削除できます。
- アカウント解約後、30日間の猶予期間を経て、全てのユーザーデータを完全に削除します。
- 削除後のデータの復元はできません。

### 4.4 セキュリティ対策

当社は、個人情報の漏洩、滅失、毀損を防止するために、以下のセキュリティ対策を実施しています。

- 通信の暗号化（TLS/SSL）
- パスワードのハッシュ化保存
- アクセス権限の管理
- Cloudflare のセキュリティ機能（WAF、DDoS 対策）の活用

## 5. Cookie の使用

### 5.1 使用する Cookie の種類

| Cookie の種類 | 目的 | 有効期間 |
|---|---|---|
| セッション Cookie | ログイン状態の維持、認証情報の管理 | ブラウザを閉じるまで、または一定期間 |
| 機能性 Cookie | ユーザーの設定・選択の保存 | 最大30日間 |

### 5.2 Cookie の管理

- 本サービスは、広告目的の Cookie やトラッキング Cookie を使用しません。
- ユーザーは、ブラウザの設定により Cookie を無効にすることができますが、その場合本サービスの一部機能が正常に動作しない可能性があります。

## 6. ユーザーの権利

ユーザーは、当社に対して以下の請求を行うことができます。

1. **開示請求**: 当社が保有するユーザーの個人情報の開示
2. **訂正請求**: 個人情報の内容が事実でない場合の訂正
3. **削除請求**: 個人情報の削除（ただし、法令に基づく保存義務がある場合を除く）
4. **利用停止請求**: 個人情報の利用停止

上記の請求は、下記のお問い合わせ先までご連絡ください。本人確認の上、合理的な期間内に対応します。

## 7. プライバシーポリシーの変更

当社は、必要に応じて本プライバシーポリシーを変更することがあります。重要な変更がある場合は、本サービス上での掲示またはメールにてお知らせします。
