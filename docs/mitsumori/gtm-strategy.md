> 作成: CMO ソフィア・リベラ

## エグゼクティブサマリー

ミツモリAI の GTM は **「見積が30分で終わる」の一点突破** で攻める。ターゲットは従業員1〜10名の小規模工務店の社長。2024年問題（残業規制）で見積作成の時間が削られている今、このメッセージは「あると便利」ではなく「ないと回らない」レベルで刺さる。

戦略の柱:
1. **X で建設業界の「見積つらい」コミュニティに入り込む**（リリース前から）
2. **YouTube デモ動画で「30分で見積完成」を視覚的に証明する**（リリース同時）
3. **SEO ブログで「工務店 見積 効率化」の検索流入を取る**（リリース前から仕込む）
4. **ウェッジ戦略で最初の10社を個別に獲得する**（リリース後2週間）

目標: リリース3ヶ月後に **MRR ¥150,000（有料15社）**

## 4. ウェッジ戦略の具体化 — 最初の10社の獲り方

### 4.1 X リプライ戦略（「見積 つらい」系へのアプローチ）

**検索監視キーワード**:
- `見積 つらい` / `見積 面倒` / `見積 時間かかる`
- `積算 人手不足` / `積算 大変`
- `工務店 効率化` / `工務店 忙しい`
- `2024年問題 残業` + 建設系ワード

**リプライの原則**:
1. **いきなり売り込まない**。まず共感 → 有益な情報提供 → 自然にプロダクト紹介
2. **3ステップ構成**:
   - Step 1（共感）: 「見積作成、本当に大変ですよね。社長が全部やるしかない規模感だと特に…」
   - Step 2（価値提供）: 「うちも同じ課題を研究していて、過去見積のパターンを AI に学習させると劇的に時短できることがわかりました」
   - Step 3（軽い誘導）: 「もし興味があれば、7日間無料で試せるツールを作ったので覗いてみてください → URL」
3. **bot っぽくしない**。投稿の具体的な内容に触れる。テンプレリプライは禁止
4. **頻度**: 1日3〜5件のリプライ。やりすぎるとスパム扱いされる

**リプライ例**:

> 投稿: 「今日も見積作成で3時間溶けた…残業規制で時間ないのに」
>
> リプライ: 「お疲れさまです。3時間は本当にきついですね…。2024年問題で残業もできない中、見積業務が一番圧迫してるという声をよく聞きます。過去の見積データをAIに学習させると、類似案件の見積を30分で出せるようになります。もしよければ試してみてください（7日間無料です） → mitsumori.agiinc.io」

### 4.2 業界コミュニティへのアプローチ

| チャネル | アクション | 期待効果 |
|---|---|---|
| 住宅リフォーム推進協議会 | 会員向けメルマガに無料トライアル案内を打診 | 一括リーチ。信頼性の担保 |

### 4.3 個別オンボーディング（最初の10社）

最初の10社は **白手袋対応** する。ここでの学びが全てのマーケ施策の素材になる。

| ステップ | 内容 | 所要時間 |
|---|---|---|
| 1. 初回ヒアリング | 現在の見積フロー、使用ツール、課題を聞く | 30分 |
| 2. データ取り込み支援 | 過去見積の CSV 変換をサポート（テンプレート提供 + 記入例） | 1時間 |
| 3. 初回見積生成 | 一緒に AI 見積を1件作成。精度を確認 | 30分 |
| 4. フィードバック収集 | 「何が良かった？何が足りない？」を記録 | 15分 |
| 5. 2週間後フォロー | 利用状況確認。有料転換の判断材料を提供 | 15分 |

**収集すべき素材**:
- 「Before/After の具体的な時間」（マーケ素材に使用）
- 「一言コメント」（X・LP のソーシャルプルーフに使用）
- 「見積精度の評価」（プロダクト改善に使用）
